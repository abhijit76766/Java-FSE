# 04 - Advanced Mockito Hands-On Exercises

Maven project (JUnit 5 + Mockito). Run with `mvn test`.

- **Exercise 1:** `Repository` + `Service` + `ServiceTest`.
- **Exercise 2:** `RestClient` + `ApiService` + `ApiServiceTest`.
- **Exercise 3:** `FileReader`/`FileWriter` + `FileService` +
  `FileServiceTest`.
- **Exercise 4:** `NetworkClient` + `NetworkService` +
  `NetworkServiceTest`.
- **Exercise 5:** chained `thenReturn()` calls, covered in
  `ServiceTest#testServiceWithMultipleReturnValues`.
