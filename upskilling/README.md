# Upskilling Solutions

Solutions for the Java FSE Upskilling module exercises.

## Folder Structure

- `module-1-html5` - HTML5 Local Community Event Portal solution.
- `module-1-css3` - CSS3 styling solution for the portal.
- `module-1-javascript` - JavaScript portal interactions and registrations.
- `module-1-bootstrap5` - Bootstrap 5 exercise page covering grid, forms, cards, icons, modal, accordion, and Sass notes.
- `module-2-ansi-sql-mysql` - MySQL schema, sample data, and all 25 SQL query solutions.
- `module-3-core-java` - Java source files for the 41 Core Java exercises.

Open the HTML folders in a browser. Run SQL scripts in MySQL. Compile Java files from `module-3-core-java/src`.
