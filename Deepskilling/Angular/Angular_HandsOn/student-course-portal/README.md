# Angular Hands-On Solution - Student Course Portal

This folder contains the cumulative solution for all 10 Angular hands-on exercises from `Angular_HandsOn.pdf`.

## Run

```bash
npm install
npm run api
npm start
```

Open `http://localhost:4200`.

## Test

```bash
npm test
npm test -- --code-coverage
```

## Covered Hands-On Items

- Project setup, notes, build budgets, routing, and core components.
- Binding, lifecycle hooks, `@Input`, `@Output`, and component communication.
- Structural directives, attribute directives, custom highlight directive, and credit label pipe.
- Template-driven and reactive forms with sync, async, and dynamic validators.
- Services, root providers, component-level providers, and hierarchical DI.
- Route parameters, query parameters, lazy enrollment routes, and guards.
- HttpClient CRUD methods, RxJS operators, retry, error handling, and interceptors.
- NgRx actions, reducers, selectors, effects, and enrollment state.
- Jasmine/Karma specs for component, service, and store-connected component testing.
