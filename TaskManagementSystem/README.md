# Task Management System

A singly linked list stores each task in a node that points to the next node. A doubly linked list also stores a previous pointer, making backward traversal easier at the cost of extra memory.

Time complexity:
- Add at end: `O(n)` in this simple implementation.
- Search: `O(n)`.
- Traverse: `O(n)`.
- Delete by id: `O(n)`.

Linked lists are useful for dynamic data because they grow without needing contiguous memory or manual capacity management.
