# React main

Solutions for the 19 ReactJS Hands-On Lab (HOL) exercises.

Each numbered folder is a standalone Create React App project
(`src/`, `public/`, `package.json`) containing the completed solution
for that lab's task, plus a README explaining the approach.

| # | Folder | Topic |
|---|--------|-------|
| 1 | 01-myfirstreact | Setting up React, create-react-app |
| 2 | 02-StudentApp | Class components (Home/About/Contact) |
| 3 | 03-scorecalculatorapp | Function component + props |
| 4 | 04-blogapp | Lifecycle: componentDidMount, componentDidCatch, fetch |
| 5 | 05-cohort-styling-app | CSS Modules, inline styling |
| 6 | 06-TrainersApp | React Router, useParams |
| 7 | 07-shoppingapp | Props vs state, class components |
| 8 | 08-counterapp | React state, constructor |
| 9 | 09-cricketapp | ES6: map, filter, arrow fns, destructuring, spread |
| 10 | 10-officespacerentalapp | JSX, inline CSS |
| 11 | 11-eventexamplesapp | Events, synthetic events, this keyword |
| 12 | 12-ticketbookingapp | Conditional rendering |
| 13 | 13-bloggerapp | Multiple conditional rendering techniques |
| 14 | 14-employee-context-app | Context API (createContext, useContext) |
| 15 | 15-ticketraisingapp | Forms, handleSubmit |
| 16 | 16-mailregisterapp | Form validation |
| 17 | 17-fetchuserapp | Consuming REST APIs |
| 18 | 18-cohort-testing-app | Unit testing with Jest + Enzyme |
| 19 | 19-gitclientapp | axios + GitHub API + Jest mocking |

## Running any exercise

```
cd <exercise-folder>
npm install
npm start
```

For exercises with tests (18, 19):
```
npm test
```
