# Exercise 9 - cricketapp

- `ListofPlayers`: array of 11 players (name + score) rendered with
  `map()`; a second list filters players with score below 70 using an
  arrow function with `filter()`.
- `IndianPlayers`: destructuring splits a team array into odd/even
  team players; two arrays (T20players, RanjiTrophyPlayers) are merged
  using the ES6 spread operator.
- App.js toggles between the two components using a simple `flag`
  variable with a ternary (if/else).
