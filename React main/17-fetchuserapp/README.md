# Exercise 17 - fetchuserapp

`Getuser` class component invokes the randomuser.me REST API using
`fetch()` inside `componentDidMount()`, then renders the user's title,
first name and image once the data arrives.

Note: the lab text references `https://api.randomuser.me/`; the
correct/working endpoint for the Random User API is
`https://randomuser.me/api/`, which is what this solution uses.
